One Liners Commands



Run The Last Command As Root

sudo !!


Serve current directory tree at http://$HOSTNAME:8000/

python -m SimpleHTTPServer


Get Your Exrernal IP Address

curl ifconfig.me


Download an Entire Website

wget --random-wait -r -p -e robots=off -U mozilla http://www.example.com


Update Twitter Via Curl

curl -u user:pass -d status="Tweeting from the shell" http://twitter.com/statuses/update.xml


Watch Starwars ViaT Telnet

telnet towel.blinkenlights.nl


Recover Delete File

grep -a -B 25 -A 100 'some string in the file' /dev/sda1 > results.txt


Extract Audio From Flash Video as mp3 file

ffmpeg -i video.flv -vn -ar 44100 -ac 2 -ab 192k -f mp3 audio.mp3


Convert Text to Picture

echo -e "Some Text Line1\nSome Text Line 2" | convert -background none -density 196 -resample 72 -unsharp 0x.5 -font "Courier" text:- -trim +repage -bordercolor white -border 3 text.gif


Check Site SSL Certificates

echo | openssl s_client -connect www.google.com:443 2>/dev/null |openssl x509 -dates -noout

Convert PDF to JPG

for file in `ls *.pdf`; do convert -verbose -colorspace RGB -resize 800 -interlace none -density 300 -quality 80 $file `echo $file | sed 's/\.pdf$/\.jpg/'`; done

Download Entire Static Website to Local Machine

wget --recursive --page-requisites --convert-links www.example.com

Download All MP3 listed in HTML Page

wget -r -l1 -H -t1 -nd -N -np -A.mp3 -erobots=off [url of website]
